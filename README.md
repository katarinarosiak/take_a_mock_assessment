# Take A Mock Assessment

- Helps you to test your knowledge about frontend JavaScript (AJAX, DOM maniupulation, API and more)
- Generates list of 20 random questions that are fetched from Notion page using Notion API
- uses Express.js server hosted on Heroku as a proxy
